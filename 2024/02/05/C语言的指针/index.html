C 语言指针入门：地址、解引用与函数参数

发表于2024-02-05|更新于2024-02-05|C语言基础
|阅读量:

刚接触单片机时，我会用指针，却一直说不清它到底是什么。后来把“变量、地址、解引用”三件事拆开看，思路才顺下来。这篇只讲最基础的部分。

指针存的是地址

普通变量保存数据，指针变量保存某个对象的地址：

这里有两个容易混淆的符号：

  • &value：取得 value 的地址；
  • *ptr：访问 ptr 所指地址里的数据，也叫解引用。

int *ptr 表示 ptr 指向 int。指针本身也是变量，也占用内存，只是它保存的内容是一段地址。
